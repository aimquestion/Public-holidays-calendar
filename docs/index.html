How to subscribe

Outlook (new / web / Microsoft 365)
  1. Copy the calendar link above.
  2. In Outlook go to Calendar → Add calendar → Subscribe from web.
  3. Paste the link, give it a name, and select Import.

Outlook refreshes internet calendars on its own schedule (often several hours to a day) — fine for dates that change once a year.

Google Calendar
  1. Copy the calendar link.
  2. Other calendars → + → From URL, paste, Add calendar.
Apple Calendar (Mac / iPhone)
  1. Tap Subscribe above, or copy the link.
  2. Mac: File → New Calendar Subscription, paste, Subscribe.
